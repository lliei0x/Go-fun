  爬取 `Github` 上的 `Trending` 开源项目以及相关作者

- [FIFA-Backstage](https://github.com/leeifme/Go-fun/tree/master/FIFA-Backstage)

  主要使用技术：

  - [x] gorm 操作数据库
  - [x] fresh 实现 web server 监听
  - [x] viper 实现读取用户配置
  - [x] 数据库 使用 postgre
  - [x] goquery 实现网页解析
  - [ ] gin 快速搭建 web server
  - [ ] gin-swagger 自动化构建 API 文档
  - [ ] jwt 认证，后台生成对应的 auth_token
